# 🏙️ 창원 중동 유니시티 매물 통합 관리 플랫폼

여여부동산에서 직접 운영하는 유니시티 아파트 매물 시각화 및 비교 분석 플랫폼입니다.

## ✨ 핵심 기능
- **정밀 슬라이더 비교**: 두 매물을 1:1로 배치하여 시각적으로 대조 분석 가능
- **스마트 필터링**: 단지, 평형, 공간별 맞춤 검색으로 고객 상담 대응력 강화
- **영상 투어**: 유튜브 연동을 통한 생생한 단지 투어 기능
- **매물 관리 시스템**: 관리자 인증을 통한 실시간 사진 및 영상 업로드

## 🚀 배포 가이드
1. **GitHub 연동**: 본 저장소를 GitHub에 업로드합니다.
2. **Vercel 배포**: Vercel에서 GitHub 저장소를 선택하기만 하면 자동으로 빌드 및 무료 배포됩니다.
3. **환경 설정**: 어떠한 API Key도 요구하지 않으므로 추가 설정 없이 즉시 사용 가능합니다.
